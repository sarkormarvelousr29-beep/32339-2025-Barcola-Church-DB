# Church Management System

## Overview
The Church Management System is an Oracle Database project developed as a capstone project for the **DPR400210 – Database Programming with Oracle Database** course. The system helps churches manage member information, ministries, donations, attendance, and church events using SQL and PL/SQL.

## Project Objectives
- Manage church member records.
- Manage ministries and member assignments.
- Record church events and attendance.
- Record tithes, offerings, and donations.
- Demonstrate Oracle SQL and PL/SQL programming concepts.
- Implement database security, auditing, and triggers.
